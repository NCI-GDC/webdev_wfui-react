# WFUI React

# Installation
```
npm install
```

# Run Component (hot reload)
```
npm start
http://192.168.59.3:9966/
```

# Build Bundle 
```
webpack
```

# Test Components
```
npm test
```



# Notes
If
```
npm install -g budo
```
and / or
```
npm install -g webpack
```
fail, try the following:
```
sudo apt-get purge nodejs npm
curl -sL https://deb.nodesource.com/setup | sudo bash -
sudo apt-get install nodejs
```
